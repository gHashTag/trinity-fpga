# tri-sim-plot — Terminal Visualization

## Назначение

ASCII-визуализатор результатов симуляции эволюции мозга. Читает CSV файл с метриками всех сценариев и отображает графики или таблицу.

## Использование

```bash
tri-sim-plot --view=MODE [--input=PATH]
```

### Режимы визуализации

| Режим | Описание |
|------|-------------|
| `--view summary` | Таблица финальных метрик, отсортированных по PPL |
| `--view ppl` | Графики эволюции PPL для всех сценариев |
| `--view diversity` | Тренды индекса разнообразия во времени |
| `--view alive` | Кривые выживаемости воркёров |

## Формат CSV

Файл `simulation_results.csv` содержит следующие столбцы (должен быть минимум для работы summary):

| Столбец | Описание |
|---------|-------------|
| step | Номер шага симуляции |
| scenario_id | Идентификатор сценария (S1-S15) |
| ppl | Перплексичность модели |
| diversity | Индекс разнообразия (0.0-0.82 = низкая, 1.0 = высокая) |
| alive | Количество живых воркёров |
| energy | Потраченная энергия (alive × step) |
| kill_threshold | Порог отключения воркёров |
| crash_rate | Частота крашей воркёров |
| byzantine_rate | Доля византийских воркёров |
| obj_ppl | Веса цели PPL |
| obj_div | Веса цели разнообразия |
| obj_weights | Веса всех целей (JSON массив) |

**Примечание:** Файл генерируется симулятором, все столбцы обязательны, но порядок столбцов может меняться.

## Цветовая кодировка сценариев

| Категория | Цвет |
|---------|--------|
| Baseline | Зелёный (default) |
| Sacred | Пурпурный |
| dePIN | Жёлтый |
| Wide | Голубой |
| Mixed | Синий |

## Команды

```bash
tri-sim-plot --view=summary --input=/tmp/simulation_results.csv
tri-sim-plot --view=ppl
tri-sim-plot --view=diversity
tri-sim-plot --view=alive
```

## Ограничения

- Требуется минимум 15 сценариев (S1-S15)
- Максимальный размер графиков — 60×20 символов
- Бинар поддерживает все 4 режима визуализации
- Целочисловое число — чистый Zig без внешних зависимостей
