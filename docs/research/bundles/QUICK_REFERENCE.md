## Bundle Dependencies

```
                    ┌─────────────┐
                    │   PARENT    │
                    │  Collection │
                    └──────┬──────┘
                           │
        ┌──────────────────┼──────────────────┐
        │                  │                  │
   ┌────▼────┐        ┌───▼────┐        ┌───▼────┐
   │  B001   │◄───────│  B006   │        │  B002   │
   │  HSLM   │        │  GF16   │        │  FPGA   │
   └────┬────┘        └─────────┘        └─────────┘
        │                  ▲
        │                  │
   ┌────▼────┐        ┌───┴────┐
   │  B007   │        │  B005  │
   │   VSA   │        │ TriLang│
   └─────────┘        └───┬────┘
                          │
                     ┌────▼────┐
                     │  B003   │
                     │ TRI-27  │
                     └────┬────┘
                          │
                     ┌────▼────┐
                     │  B004   │
                     │  Lotus  │
                     └─────────┘
```

**Legend:**
- `──►` : Uses/Implements
- `◄────` : Depends on
- B001 uses B006 (GF16 encoding) and B007 (VSA acceleration)
- B002 can run B001 models
- B005 compiles to B002 targets
- B003 is the ISA for B005 compilation
- B004 manages the overall agent lifecycle
