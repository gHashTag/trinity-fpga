## Custom Badge Parameters

### Color Options

```markdown
# Named colors
?color=brightgreen
?color=green
?color=yellowgreen
?color=yellow
?color=orange
?color=red
?color=pink
?color=purple
?color=blue
?color=blueviolet

# Hex colors
?color=209cff  # Zenodo blue
?color=3498db  # Trinity blue
```

### Logo Options

```markdown
?logo=data:image/png;base64,...
?logo=zenodo
&logoWidth=20
```

### Label Options

```markdown
?label=zenodo
?label=DOI
?label=BUNDLE
?message=10.5281/zenodo.19227879
```
