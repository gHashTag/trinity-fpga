# 🚀 Quick Start: Compile Paper on Overleaf

## Option 1: Overleaf (RECOMMENDED — Fast & Easy)

1. **Go to:** https://www.overleaf.com/

2. **Create new project:** Click "New Project" → "Blank Paper"

3. **Replace main.tex with our paper:**
   - Delete everything in main.tex
   - Copy paste from: `/Users/playra/trinity-w1/docs/papers/trinity-sacred-mathematics.tex`

4. **Click "Recompile"** — PDF will be generated automatically!

5. **Download:** Click "PDF" → "Download PDF"

---

## Option 2: Install MacTeX Locally (Requires Password)

Open your terminal and run:

```bash
brew install mactex-no-gui
```

**When asked:** Enter your macOS password

**After installation:** Restart terminal and run:

```bash
cd /Users/playra/trinity-w1/docs/papers
pdflatex trinity-sacred-mathematics.tex
```

## Option 3: Use Online LaTeX Compiler (No Sign-up)

1. Go to: https://www.overleaf.com/latex/latex_compiler
2. Copy content from: `trinity-sacred-mathematics.tex`
3. Paste into the online editor
4. Click "Compile"
