Official Homebrew tap for Trinity — Ternary Vector Symbolic Architecture.

## Installation

```bash
# Add the tap
brew tap ghashtag/trinity

# Install Trinity
brew install trinity
```

## What's Installed

- **tri** — Unified Trinity CLI (195+ commands)
- **vibee** — VIBEE specification compiler
- **firebird** — BitNet-to-Ternary inference engine

## Quick Start

```bash
# Show all commands
tri --help

# Run sacred mathematics
tri phi 10

# Generate code from spec
tri gen my_spec.vibee

# Start chat mode
tri chat
```
