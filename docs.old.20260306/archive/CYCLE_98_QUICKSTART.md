## COMMAND REFERENCE

### Development Commands

```bash
# Create specs
tri spec-create <name>                    # Create .vibee spec template

# Generate code
tri gen specs/tri/<name>.vibee           # Generate Zig code

# Test
zig test src/<module>.zig                 # Test single module
zig build test                            # Run all tests

# Run evolution
tri evolve                                # Start manual evolution cycle
tri daemon start                          # Start eternal daemon
tri daemon stop                           # Stop daemon

# Check governance
tri govern check <file>                   # Check sacred compliance
tri govern enforce                        # Enable auto-enforcement

# Swarm operations
tri swarm spawn <agent_type>              # Spawn new agent
tri swarm status                          # Show swarm state
tri swarm vote <proposal>                 # Cast vote

# Dashboard
tri dashboard serve                       # Start dashboard dev server
tri dashboard deploy                      # Deploy to gh-pages
```
