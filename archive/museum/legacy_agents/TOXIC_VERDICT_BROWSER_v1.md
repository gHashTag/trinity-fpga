## AI AGENT FRAMEWORK

**Аon[CYR:[TRANSLATED]] Eko from Fellou, но for VIBEE:**

```javascript
const agent = new VIBEEAgent({
    model: 'gpt-4',
    maxSteps: 20,
    browser: browserInterface,
    llm: llmInterface,
    verbose: true
});

// Run agent with goal
const result = await agent.run('Go to google.com and search for VIBEE');

// Agent uses ReAct pattern:
// 1. Think: Reason about current state
// 2. Act: Execute browser action
// 3. Observe: Check result
// 4. Repeat until done
```

**Available Tools:**
- `navigate` - Go to URL
- `click` - Click element
- `type` - Type text
- `scroll` - Scroll page
- `screenshot` - Take screenshot
- `extract` - Extract text
- `getPageContent` - Get page text
- `wait` - Wait for time
- `done` - Complete task
