## Roles

### Everyone
- Default role for all members

### Verified Developer
- Assigned after first PR merged
- Can post in #development
- Access to #pr-reviews

### Contributor
- 5+ merged PRs
- Can triage issues
- Access to contributor channels

### Researcher
- Published paper on VSA/ternary
- Access to #papers
- Special role in research channels

### Core Team
- Project maintainers
- Full admin access
