## COMPETITOR ANALYSIS

### рҹҘҮ Fellou (Best-in-class Agentic)
**Strengths:**
- Full browser automation (click, type, navigate)
- Multi-agent background tasks
- Cross-site workflows
- Agentic memory
- Deep research reports
- Eko 2.0 open-source framework

**Weaknesses:**
- No local LLM support
- No privacy features (ad/tracker blocking)
- No voice interaction
- Closed-source browser core

### рҹҘҲ Dia Browser (Arc successor)
**Strengths:**
- Tab-aware context
- Writing assistance
- Learning/tutoring features
- Clean UI from Arc team

**Weaknesses:**
- No automation
- No agent reasoning
- macOS only (M1+)
- Limited features

### рҹҘү Brave Leo (Privacy-first)
**Strengths:**
- Multiple LLM models
- BYOM (Bring Your Own Model)
- Local processing option
- No account required
- Built-in VPN, ad blocking

**Weaknesses:**
- No browser automation
- No agent reasoning
- No multi-tab orchestration

### Opera Aria
**Strengths:**
- Tab Island context
- Image generation
- Voice I/O
- YouTube integration
- Free VPN

**Weaknesses:**
- No automation
- No agent reasoning
- Privacy concerns

### Edge Copilot
**Strengths:**
- Deep Microsoft integration
- Email/calendar commands
- Enterprise features

**Weaknesses:**
- Microsoft lock-in
- No automation
- Privacy concerns
