### Next Steps (To Complete BENCH-005)

1. **Build nextpnr-xilinx**:
   ```bash
   cd fpga/nextpnr-xilinx
   cmake ..
   make
   ```

2. **Run place & route**:
   ```bash
   nextpnr-xilinx --chipdb ../openxc7-synth/chipdb/xc7a100tfgg676.bin \
                   --xdc ../openxc7-synth/gf16_top.xdc \
                   --json gf16_add_top.json \
                   --fasm gf16_add_top.fasm
   ```

3. **Convert to bitstream**:
   ```bash
   python3 fpga/prjxray/utils/fasm2frames.py \
       --db-root fpga/prjxray/database/artix7 \
       --part xc7a100tfgg676-1 \
       --sparse gf16_add_top.fasm \
       gf16_add_top.frames
   xc7frames2bit --part_file fpga/prjxray/database/xc7a100tfgg676-1/part.yaml \
                   --part_name xc7a100tfgg676 \
                   --frm_file gf16_add_top.frames \
                   --output_file gf16_add_top.bit
   ```

4. **Repeat for multiplier** (`gf16_mul_top`)

5. **Extract Fmax** from nextpnr timing report (`*.rpt`)

6. **Optional: Flash and verify LED behavior**:
   ```bash
   tri fpga flash fpga/openxc7-synth/gf16_add_top.bit
   tri fpga eye snap
   ```

7. **Update documentation** with Fmax values
